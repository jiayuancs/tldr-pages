# man [modified]

> Format and display manual pages.
> More information: <https://manned.org/man>.

- Display the man page for a command:

`man {{command}}`

- Display the man page for a command from section 7:

`man {{7}} {{command}}`

- 查找，等同于 whatis 命令：

`man -f {{command}}`

- 模糊查找，等同于 apropos 命令：

`man -k apropos`
