# cat [modified]

> Print and concatenate files.
> More information: <https://www.gnu.org/software/coreutils/cat>.

- Print the contents of a file to the standard output:

`cat {{path/to/file}}`

- 显示行号：

`cat -n {{path/to/file}}`

- 显示行号，但空白行不标行号：

`cat -b {{path/to/file}}`

- 拼接多个文本文件：

`cat {{path/to/file1 path/to/file2 ...}} > {{path/to/output_file}}`
