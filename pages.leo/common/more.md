# more [modified]

> Open a file for interactive reading, allowing scrolling and search.
> More information: <https://manned.org/more>.

- Open a file:

`more {{path/to/file}}`

- 下一页：

`<Space>`

- 上一行/下一行：

`↑ / ↓`

- 搜索字符串 (press `n` to go to the next match):

`/{{something}}`

- 退出：

`q`

- 查看帮助信息：

`h`
