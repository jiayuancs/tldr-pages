# git [modified]

> Distributed version control system.
> Some subcommands such as `commit`, `add`, `branch`, `checkout`, `push`, etc. have their own usage documentation, accessible via `tldr git subcommand`.
> More information: <https://git-scm.com/>.

- 创建、删除分支

`tldr git-branch`

- 切换分支

`tldr git-checkout`

- 版本回退

`tldr git-reset`
