# bash shortcut keys [new]

> bash shortcut keys

- 删除光标左边/右边的一个单词

`Ctrl-w / Alt-d`

- 删除光标左边/右边的所有内容

`Ctrl-u / Ctrl-k`

- 删除光标左边/右边的一个字符

`Ctrl-h / Ctrl-d`

- 光标移动到行首/行尾

`Ctrl-a / Ctrl-e`

- 光标向左([b]ackword)/向右([f]orward)移动一个字符

`Alt-b / Alt-f`

- 显示上一条/下一条历史命令

`Ctrl-p / Ctrl-n`

- 使用默认编辑器编辑命令(使用export EDITOR=vim设置vim为默认编辑器)

`Ctrl-x Ctrl-e`

- 中断，结束当前任务

`Ctrl-c`

- 将当前任务放到后台，状态为Stopped，可使用jobs命令查看

`Ctrl-z`

- 有多重含义，在处理标准输入时表示EOF，其他情况通常相当于执行exit命令

`Ctrl-d`

- 清空屏幕，相当于clear

`Ctrl-l`
